## Featured project

### Temperature Anomaly Analysis (1970–2024)

A statistical analysis of temperature anomaly trends in the **Canary Islands** in comparison with **global** and **hemispheric series**, based on data from **AEMET** and **HadCRUT5**.

#### Main highlights
- Linear trend estimation using **OLS regression**
- Robust inference with **HAC / Newey-West confidence intervals**
- Non-parametric trend analysis with **Mann-Kendall** and **Sen's slope**
- Multi-scale study using **monthly**, **annual**, and **month-by-month** series
- Comparison with alternative methods such as **Yeo-Johnson transformation** and **segmented regression**
- Evaluation of recent trend behavior and model reliability

📌 **Accepted at the XIV Congreso de la Asociación Española de Climatología (AEC 2026)**
